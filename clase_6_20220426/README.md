# Comandos de la consola utilizados en la clase

## Limpiar consola
```clear```

## Cambiar carpeta
```cd .\clase_5_20220421```

## Crear proyecto de nodojs
```npm init -y```


## Correr archivos con extension .js
```node .\archivo.js```

## Como instalar dependencia express
```npm install express```
